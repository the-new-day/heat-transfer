# Тепловая песочница — энтальпийная модель фазового перехода

## Физические основы

### 1. Уравнение теплопроводности

В основе симуляции лежит двумерное уравнение теплопроводности (закон Фурье):

$$\rho c_p \frac{\partial T}{\partial t} = \nabla \cdot (k \nabla T) + q$$

где:
- $T$ — температура (°C)
- $\rho$ — плотность вещества (кг/м³)
- $c_p$ — удельная теплоёмкость (Дж/(кг·К))
- $k$ — коэффициент теплопроводности (Вт/(м·К))
- $q$ — объёмная мощность источника тепла (Вт/м³)

Уравнение решается на равномерной прямоугольной сетке 60×60 ячеек методом конечных разностей с явной схемой по времени (метод Эйлера).

### 2. Энтальпийный метод — учёт фазового перехода

При плавлении вещество поглощает скрытую теплоту без изменения температуры. Чтобы корректно отобразить этот процесс, вместо температуры в качестве основной переменной используется **удельная энтальпия** $H$ (Дж/кг):

$$H = \begin{cases}
c_p T, & T < T_\text{пл} - \delta \\
c_p T + L \cdot \varphi(T), & T_\text{пл} - \delta \leq T \leq T_\text{пл} + \delta \\
c_p T + L, & T > T_\text{пл} + \delta
\end{cases}$$

где $L$ — удельная теплота плавления (Дж/кг), $\delta$ — полуширина переходной зоны (°C).

Обратный пересчёт $H \to T$ позволяет избежать разрыва в температурном поле. Уравнение, решаемое численно:

$$\frac{\partial H}{\partial t} = \frac{1}{\rho} \left[ \nabla \cdot (k \nabla T) + q \right]$$

Этот подход называется **методом энтальпии** (enthalpy method) и широко применяется в задачах типа Стефана.

### 3. Доля жидкой фазы и «мягкая» зона плавления

Мгновенный фазовый переход создаёт численные сложности. Реальные вещества плавятся в диапазоне температур (мягкая зона Mushy zone). Доля расплавленного вещества в ячейке:

$$\varphi = \mathrm{clamp}\!\left(\frac{T - (T_\text{пл} - \delta)}{2\delta},\ 0,\ 1\right)$$

При $\varphi = 0$ вещество полностью твёрдое, при $\varphi = 1$ — полностью жидкое. Теплопроводность в переходной зоне интерполируется линейно:

$$k(T) = k_\text{тв} + (k_\text{ж} - k_\text{тв})\, \varphi(T)$$

### 4. Гармоническое среднее на границах ячеек

При вычислении теплового потока между двумя соседними ячейками с разными $k$ используется **гармоническое среднее**:

$$k_{ij} = \frac{2\, k_i\, k_j}{k_i + k_j}$$

Это физически правильная формула для последовательного соединения тепловых сопротивлений (аналогично закону Ома для последовательно включённых резисторов). Она автоматически корректно обрабатывает границу изолятор–проводник.

### 5. Дискретизация и устойчивость

Дивергенция потока для ячейки $(x, y)$:

$$(\nabla \cdot k \nabla T)_{x,y} = \frac{k_{x+\frac{1}{2}} (T_{x+1} - T_x) - k_{x-\frac{1}{2}} (T_x - T_{x-1})}{\Delta x^2} + \frac{k_{y+\frac{1}{2}} (T_{y+1} - T_y) - k_{y-\frac{1}{2}} (T_y - T_{y-1})}{\Delta y^2}$$

Шаг по времени ограничен условием устойчивости явной схемы:

$$\Delta t \leq \frac{\Delta x^2}{4\, \alpha_\text{max}}, \quad \alpha = \frac{k}{\rho c_p}$$

Граничные условия — адиабатические (нулевой поток через стенки).

### 6. Источники тепла

Мощность, вводимая в ячейку от точечного источника, описывается гауссовой функцией:

$$q(x, y) = P \cdot \exp\!\left(-\frac{r^2}{2\sigma^2}\right) \cdot \tau(x, y)$$

где $P$ — мощность источника, $\sigma = 5$ мм — характерный радиус, $r$ — расстояние от источника. Коэффициент $\tau(x, y)$ учитывает экранирование изоляторами вдоль прямой между источником и ячейкой (трассировка луча):

$$\tau = \begin{cases} 1, & \text{нет изоляторов на пути} \\ 0.03^n, & n \text{ ячеек изолятора на пути} \end{cases}$$

### 7. Используемые материалы

| Материал | $\rho$ (кг/м³) | $c_p$ (Дж/(кг·К)) | $k_\text{тв}$ (Вт/(м·К)) | $k_\text{ж}$ (Вт/(м·К)) | $L$ (кДж/кг) | $T_\text{пл}$ (°C) |
|----------|----------------|-------------------|--------------------------|--------------------------|---------------|---------------------|
| Водяной лёд | 917 | 2090 | 2.22 | 0.58 | 334 | 0.0 |
| Парафин | 900 | 2100 | 0.24 | 0.15 | 200 | 54.0 |
| Галлий | 5910 | 370 | 32.0 | 29.0 | 80 | 29.8 |

Параметры изолятора: $\rho = 500$ кг/м³, $c_p = 2000$ Дж/(кг·К), $k = 0.045$ Вт/(м·К).

### 8. Поиск оптимального положения

Алгоритм определяет позицию, в которой объект тает медленнее всего. Для этого численно решается вспомогательная задача: диффузия тепла без объекта на 500 шагов с увеличенным $\Delta t$ (устойчивое значение). Полученное поле фоновой температуры $T_\text{bg}$ отражает реальный нагрев каждой точки поля с учётом геометрии изоляторов. Выбирается позиция с минимальной средней $T_\text{bg}$ по области объекта (при условии отсутствия перекрытия с изоляторами).

## Описание приложения

Интерактивная симуляция позволяет наблюдать за таянием объекта в тепловом поле в реальном времени.

**Инструменты:**
- **Источник** — точечный нагреватель с гауссовым полем мощности; клик повторно удаляет источник
- **Объект** — перетаскиваемый объект из выбранного материала; подсвечивается красным, если позиция перекрывается с изолятором
- **Изолятор** — материал с низкой теплопроводностью; блокирует распространение тепла от источников
- **Стереть** — удаляет изоляторы и охладители

**Параметры:**
- Выбор материала объекта (лёд, парафин, галлий)
- Форма объекта (квадрат, круг)
- Мощность источников тепла
- Начальная температура среды (dT от температуры плавления)
- Размер объекта
- Радиус кисти для рисования изоляторов

**Управление симуляцией:**
- Пуск / Пауза / Шаг / Ускорение (1×, 4×, 12×)
- Перезапуск (сброс температур, сохранение сцены)
- Очистить сцену — полный сброс
- **Найти оптимум** — автоматически переместить объект в положение, при котором он тает медленнее всего

**Визуализация:**
- Тепловое поле с билинейной интерполяцией и цветовой шкалой
- Контур объекта (marching squares) исчезает по мере таяния
- График «масса объекта / время» с маркером момента полного расплавления
